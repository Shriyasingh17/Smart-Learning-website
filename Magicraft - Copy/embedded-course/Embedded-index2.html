Embedded Systems

An embedded system is a computer system—a combination of a computer processor, computer memory, and input/output peripheral devices—that has a dedicated function within a larger mechanical or electronic system.

What you'll learn ?

  1. Understand Right ways of Handling and programming MCU Peripherals
  2. Develop Peripheral drivers for your Microcontroller
  3. Understand complete Driver Development steps right from scratch for GPIO,SPI,I2C and USART.
  4. Learn Writing peripheral driver headers, prototyping APIs and implementation
  5. Explore MCU data sheets, Reference manuals, start-up Codes to get things done
  6. Learn Right ways of handling/configuring Interrupts for various peripherals
  7. Learn about Peripheral IRQs/Vector table/NVIC interfaces and many
  8. Guaranteed learning in every minute!

Requirements

  • Basic knowledge of C programming
  • Basic Python knowledgeIf you are completely new to MCU and C programming language then you are advised to finish our "Embedded C" course for absolute beginners first
